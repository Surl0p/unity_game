# Unity Platformer
## What is this ? 
Small 2D Platformer game made on Unity.
## Controls
W,A,S,D - Move

Space - Jump

Left click - Fire

Escape - Pause menu

## Description
"Gnome in the Forest" is a mobile arcade game in the runner genre with platformer elements, where the player controls a little gnome who runs through a magical forest, avoiding obstacles and collecting power—ups.
The game does not have well-defined levels in the traditional sense. Her goal is to go as far as possible, increasing her personal best with each new launch. The forest in which the action takes place remains visually unchanged, but the complexity of the game gradually increases over time, offering the player more and more difficult obstacles and challenges. This creates a sense of endless adventure and motivates players to re-play.
The game is aimed at both children and adults, providing intuitive controls and an endless, addictive gameplay.
